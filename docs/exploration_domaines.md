## Pistes de jeux — 6 concepts

### 1. Explorateur Spatial (aventure / exploration)
**Pitch** : Pilote un vaisseau a travers le systeme solaire. Chaque planete = un monde avec des enigmes.
- **Maths** : calculs de trajectoire (additions, multiplications), mesures de distances, fractions (carburant)
- **Sciences** : systeme solaire, gravite, matiere (etats solide/liquide/gaz sur differentes planetes)
- **Francais** : journal de bord a rediger, dialogues avec des aliens (vocabulaire)
- **Histoire** : decouvertes scientifiques reelles en bonus
- **Gameplay** : exploration 3D, puzzles physiques, collecte de ressources
- **Vibe** : No Man's Sky version enfant

### 2. Batisseuse de Civilisation (gestion / strategie)
**Pitch** : Construis et fais evoluer un village a travers les epoques (prehistoire → moyen-age → moderne).
- **Histoire** : periodes historiques, inventions, modes de vie
- **Maths** : gestion de ressources (compter, echanger, proportions), geometrie (constructions)
- **Sciences** : materiaux, agriculture, energie, cycle de l'eau
- **Francais** : lecture de parchemins/quetes, nommage des batiments
- **EMC** : cooperation entre villageois, regles de vie en communaute
- **Gameplay** : construction, gestion de ressources, quetes narratives
- **Vibe** : Animal Crossing x Age of Empires simplifie

### 3. Labo Vivant (simulation / sandbox)
**Pitch** : Gere un ecosysteme (foret, ocean, ou les deux). Fais vivre des animaux, plantes, et observe les interactions.
- **Sciences** : chaines alimentaires, cycle de vie, saisons, biodiversite, photosynthese
- **Maths** : comptage de populations, graphiques, proportions
- **Francais** : fiches encyclopediques a completer, carnet de naturaliste
- **Questionner le monde** : observation, experimentation, cause/consequence
- **Gameplay** : sandbox creatif, objectifs d'equilibre, mode "decouverte" libre
- **Vibe** : Eco (Strange Loop Games) version enfant

### 4. Chrono-Voyageuse (aventure narrative / point & click 3D)
**Pitch** : Une fille voyage dans le temps pour reparer des anomalies historiques. Chaque niveau = une epoque.
- **Histoire** : immersion dans les epoques (Egypte, Moyen-Age, Renaissance, epoque moderne)
- **Francais** : dialogues, indices a lire, enigmes de vocabulaire/conjugaison
- **Maths** : enigmes logiques, codes, mesures anciennes vs modernes
- **Geographie** : cartes du monde a differentes epoques
- **EMC** : comprendre les modes de vie, tolerance, evolution des droits
- **Gameplay** : exploration, dialogues a choix, puzzles contextuels
- **Vibe** : Professeur Layton x Carmen Sandiego

### 5. Architecte Magique (puzzle / construction 3D)
**Pitch** : Construis des structures impossibles en 3D pour resoudre des puzzles. La magie = les regles de geometrie.
- **Maths** : geometrie (formes, angles, symetrie, aires, perimetres), calcul mental (debloquer des sorts)
- **Sciences** : gravite, equilibre, materiaux, forces
- **Francais** : instructions a lire, sorts a formuler correctement (orthographe)
- **Arts** : creativite, couleurs, composition
- **Gameplay** : construction 3D, puzzles de physique, mode creatif libre
- **Vibe** : Minecraft x Monument Valley x Fantastic Contraption

### 6. Detectives des Mots (aventure / RPG)
**Pitch** : RPG ou la magie est basee sur les mots. Conjuguer = lancer des sorts. Vocabulaire = inventaire.
- **Francais** : conjugaison (sorts), vocabulaire (inventaire), grammaire (combinaisons), lecture (quetes)
- **Maths** : points de vie, calculs de degats, gestion d'inventaire
- **Logique** : enigmes, deduction
- **Gameplay** : combats au tour par tour (conjuguer pour attaquer), exploration, quetes
- **Vibe** : Scribblenauts x Pokemon x Ni no Kuni
